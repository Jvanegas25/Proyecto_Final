# Proyecto_Final
Proyecto final

##Documentacion de la API

Este proyecto consiste en el desarrollo de una API REST para la gestión de clientes y pedidos.

La API permite realizar operaciones CRUD mediante el uso de métodos HTTP como GET, POST, PUT, PATCH y DELETE.

### Endpoints principales

- /api/clientes/
- /api/pedidos/

## Donde cada endpoint permite interactuar con los recursos del sistema en formato JSON.
